# Transcribely Installation Guide

## Download Links

Choose the appropriate installer for your operating system:

### Windows

1. **Windows (64-bit Intel/AMD)** - Most Windows PCs
   - File: `Transcribely-Setup-x64.exe`
   - For: Windows 10/11 (64-bit)
   - Size: ~80MB

2. **Windows ARM64** - ARM-based Windows devices
   - File: `Transcribely-Setup-arm64.exe`
   - For: Windows 10/11 on ARM (e.g., Surface Pro X)
   - Size: ~82MB

### macOS

- **Universal macOS** - Works on all modern Macs
  - File: `Transcribely-1.0.0-universal.dmg`
  - For: macOS 10.12 or later (Intel & Apple Silicon)
  - Size: ~174MB

## Installation Instructions

### Windows Installation

1. Download the appropriate installer:
   - Regular PC → `Transcribely-Setup-x64.exe`
   - ARM device → `Transcribely-Setup-arm64.exe`
2. Double-click the installer file
3. If you see a security warning, click "More info" and then "Run anyway"
4. Follow the installation wizard:
   - Choose installation location (or use default)
   - Select additional options if desired
   - Click "Install"
5. Once installation completes, launch Transcribely from:
   - Start Menu
   - Desktop shortcut (if created during installation)
   - Search for "Transcribely"

### macOS Installation

1. Download `Transcribely-1.0.0-universal.dmg`
2. Double-click the DMG file to open it
3. Drag the Transcribely app to your Applications folder
4. First Launch:
   - Go to Applications folder
   - Right-click (or Control-click) Transcribely
   - Select "Open"
   - Click "Open" in the security dialog
5. Subsequent launches:
   - Open from Applications folder
   - Use Spotlight (Command + Space, type "Transcribely")
   - Add to Dock by dragging from Applications

## System Requirements

### Windows
- Windows 10 or Windows 11
- 4GB RAM minimum (8GB recommended)
- 200MB free disk space
- Internet connection for AWS services

### macOS
- macOS 10.12 or later
- Intel or Apple Silicon processor
- 4GB RAM minimum (8GB recommended)
- 200MB free disk space
- Internet connection for AWS services

## Troubleshooting

### Windows Common Issues

1. **Security Warning**
   - This is normal for new applications
   - Click "More info" and then "Run anyway"

2. **Installation Failed**
   - Run installer as Administrator
   - Temporarily disable antivirus
   - Ensure adequate disk space

### macOS Common Issues

1. **"App is damaged" Message**
   - Right-click the app and select "Open"
   - This only needs to be done once

2. **"Unidentified Developer" Warning**
   - Right-click the app and select "Open"
   - In System Settings → Security & Privacy, click "Open Anyway"
